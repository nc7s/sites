## 代码

行内代码已经展示过了：`const x = 42`。以下是各种语言的代码块。

### JavaScript

```javascript
async function fetchData(url) {
	const response = await fetch(url)
	if (!response.ok) {
		throw new Error(`HTTP ${response.status}`)
	}
	return response.json()
}
```

### CSS

```css
article main pre {
	padding: 1.2em 1.4em;
	border-radius: 6px;
	overflow-x: auto;
	font-size: 0.85em;
}
```

### Bash

```bash
#!/bin/bash
for file in *.md; do
	echo "Processing: $file"
	wc -w "$file"
done
```

### Rust

```rust
fn fibonacci(n: u64) -> u64 {
	match n {
		0 => 0,
		1 => 1,
		_ => fibonacci(n - 1) + fibonacci(n - 2),
	}
}
```
